# 🚌 Logística y Transporte Montes de Oca (LyT M.D.O)

Plataforma web corporativa y de servicios logísticos desarrollada con un diseño moderno, minimalista y de alta gama inspirado en la línea estética de Apple. El sitio está optimizado para ofrecer una experiencia de usuario fluida, interactiva y totalmente responsiva en cualquier dispositivo.

---

## 🚀 Stack Tecnológico y Herramientas

La arquitectura frontend de este proyecto se construyó utilizando tecnologías web nativas y estándares de la industria, garantizando un rendimiento óptimo sin dependencias innecesarias de pesados frameworks:

* **HTML5 Semantic Markup:** Estructura web limpia, accesible y optimizada para motores de búsqueda (SEO).
* **CSS3 & Modern Layouts:** 
  * Uso de **CSS Custom Properties (Variables)** para la gestión dinámica de temas (Modo Claro / Modo Oscuro).
  * **Flexbox & CSS Grid** para la distribución responsiva de componentes y tarjetas.
  * Efectos visuales de alta fidelidad como `backdrop-filter: blur(20px)` para la barra de navegación superior.
* **JavaScript (ES6+):** 
  * Manipulación del DOM en tiempo real.
  * Sistema de persistencia de preferencias de usuario mediante `localStorage` para el selector de temas.
  * Validación reactiva de formularios con retroalimentación visual inmediata.
* **Tipografía Tipográfica:** Integración de la fuente tipográfica global **Inter** vía Google Fonts.

---

✨ Características Principales
Identidad Visual Corporativa: Combinación única entre la modernidad del diseño tecnológico actual y un toque histórico tradicional representado a través de un retrato artístico estilo 1940 en color sepia.

Modo Claro / Modo Oscuro Nativo: Sistema automático de alternancia de temas controlado mediante variables CSS que respeta las preferencias del usuario.

Cotizador Interactivo en Tiempo Real: Formulario inteligente con validaciones de campos vacíos y errores, capaz de procesar estimaciones logísticas al instante.

Diseño 100% Responsivo: Adaptación fluida mediante Media Queries optimizadas para computadoras de escritorio, tablets y dispositivos móviles.

Canal de Comunicación Directa: Botón flotante integrado de acceso directo a WhatsApp para atención comercial inmediata.
